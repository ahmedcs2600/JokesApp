## Built With

- Kotlin
- ViewModel and LiveData
- Dependency Injection (Hilt)
- DataStore Preferences for caching
- BuildSrc for Dependency Managment
- Multi Module App (domain, data, cache, presentation)
- Retrofit
- ViewBinding
- Unit testing (MockK and Junit)
